# AI Agent & 后端架构

## 技术栈

### 后端开发
- **语言：** Python
- **框架：** Django, FastAPI, Django REST Framework
- **异步与任务处理：** Celery, Redis
- **数据库：** MySQL, PostgreSQL, Redis
- **容器化与部署：** Docker, Kubernetes, Nginx
- **云平台：** 阿里云

### AI & Agent 系统
- **框架：** LangChain, LangGraph
- **API：** OpenAI, LangChain
- **文档生成与检索增强：** RAG
- **Agent 内存与上下文管理**

### 全栈技术
- **前端：** React, Vue.js
- **数据库：** MongoDB, MySQL

---

## 项目与技术探索

### **AI Agent & 智能系统**
- 构建 **AI Agent 系统**，支持自动化决策与任务管理
- 设计 **多智能体协作** 系统，提升任务执行的智能性和效率
- 集成 **工具调用机制**，通过 Agent 自动调度并管理外部 API

### **后端架构与微服务**
- 使用 **异步处理** 和 **事件驱动架构** 打造高并发、低延迟的后端系统
- 实现基于 **Docker & Kubernetes** 的微服务部署，确保服务的可扩展性和高可用性

### **LLM 应用与优化**
- 探索 **RAG（检索增强生成）** 技术在 LLM 中的应用，提升 AI 输出质量
- 持久化 **Agent 上下文与记忆**，让 AI 更加智能化和个性化
